# NodJs - Express  - MONGODB- Unit Test -Swagger -OpenAPI



## Packages Installed
- Nodejs  14.17.6
- npm 7.21.1
- nodemon 2.0.16


## Usage Server

- Clone/Download a repo 
- Run `cd server`
- Run `cp .env.sample .env` and update MONGO DB credentials
- Run `npm i`
- Install nodemon by `npm install -g nodemon`
- Run `nodemon`

- run `npm run swagger-autogen`
